# MovieAPI-App

Movie API app is the app built with node.js using express and Open Movie Data Base Movie API

## Build Setup

``` bash
# install dependencies
$ sudo npm install -g --force nodemon
$ npm install

# serve with hot reload at localhost:3000
$ nodemon app.js
```
